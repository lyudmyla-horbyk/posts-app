# posts-app
This project is a simple posts-app.

### Technologies
Project is created with: 
* Vue.js
* Vuex
* Sass
* Vue-router
* Axios

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve

### Compiles and minifies for production

npm run build
```
